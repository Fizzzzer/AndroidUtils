# AndroidUtils
整理项目常用的各种工具类
# How to 
先在 build.gradle(Project:XXXX) 的 repositories 添加:

    allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
# 添加依赖
	compile 'com.github.lvfaqiang:AndroidUtils:VERSION_CODE'
	
最新[版本号](https://github.com/lvfaqiang/AndroidUtils/releases)戳这里
